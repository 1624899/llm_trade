# 贡献指南

感谢你愿意参与 LLM-TRADE。当前项目为社区版，欢迎贡献代码、文档、测试、数据源适配、前端体验和使用反馈。

## 适合贡献的方向

- 修复 Bug 和兼容性问题。
- 补充测试和复现用例。
- 改进工作台前端体验。
- 增加数据源降级、缓存和异常提示。
- 优化 README、使用说明和部署文档。
- 增加报告模板、Prompt 模板和复盘流程。

## 提交 Issue

反馈问题时建议包含：

- 操作系统和 Python 版本。
- 执行命令。
- 相关配置片段，注意隐藏 API Key。
- 完整报错或日志片段。
- 是否能稳定复现。

## 提交 PR

提交 PR 前建议先执行：

```bash
python -m unittest discover -s test
```

如果只改了部分模块，也可以执行 README 中列出的单项测试。

## 风险边界

请不要提交承诺收益、确定性荐股、代客交易或绕过数据源授权的功能。项目定位是研究、复盘和工程实验工具。
