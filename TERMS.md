# 用户协议草案

本协议是 LLM-TRADE 社区版用户协议草案。正式发布前可按实际开源许可证、发布平台和社区运营方式继续调整。

## 1. 服务定位

LLM-TRADE 提供本地数据整理、策略筛选、回测复盘、观察仓、模拟交易、报告生成和可视化工作台能力。项目定位为社区版投研效率工具，不提供个性化证券投资顾问服务，不承诺投资收益，不代客交易。

## 2. 用户责任

用户应自行判断系统输出的准确性、完整性和适用性。用户不得将系统输出作为唯一投资依据，不得要求系统或经营者承担真实交易盈亏责任。

用户应确保其使用的数据源、API Key、模型服务和外部接口具有合法授权，并遵守对应服务条款。

## 3. 开源与社区使用

社区版核心功能开放使用。用户应遵守项目许可证和第三方依赖、数据源、模型服务的许可条款。

未经许可，用户不得将项目名称、开发者身份或社区版本包装为官方荐股服务，不得对外承诺收益或误导他人认为本项目提供投资顾问服务。

## 4. 数据与内容

系统可能在本地保存行情数据、财务数据、观察仓、模拟交易流水、运行日志、报告和配置文件。用户应自行备份重要数据，并妥善保管 API Key 与授权文件。

维护者可以在用户主动提交 Issue、讨论或问题反馈时查看用户提供的软件版本、错误日志、配置摘要和复现步骤。请勿公开提交 API Key、真实账号密码或其他敏感信息。

## 5. 禁止行为

用户不得利用本系统从事违法违规活动，不得以系统名义对外承诺收益，不得将系统输出包装成确定性荐股、内幕消息或保证盈利服务。

## 6. 免责声明

因市场波动、数据错误、接口中断、模型误判、用户配置错误、网络异常或第三方服务变更导致的损失，除法律另有强制规定外，项目维护者不承担投资损失赔偿责任。

## 7. 协议变更

维护者可根据项目迭代、合规要求和社区运营方式调整本协议。重大变更应通过 README、发布说明或其他合理方式通知用户。
