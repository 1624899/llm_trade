# 社区支持

LLM-TRADE 社区版不通过 token 锁核心功能。公开仓库以社区协作、问题反馈和贡献说明为主，也接受自愿支持帮助项目持续维护。

## 参与方式

| 类型 | 说明 |
| --- | --- |
| Star | 最简单也最重要的支持，方便更多人发现项目 |
| Issue | 反馈 Bug、使用问题、数据源异常和功能建议 |
| PR | 贡献代码、文档、测试、前端体验和数据源适配 |
| 文档 | 补充安装说明、常见问题、截图和教程 |
| 测试 | 补充复现用例和自动化测试 |

## 支持我们

支持是自愿帮助社区版持续维护的方式。社区版核心功能保持开放，支持者权益围绕 LLM-TRADE 自身的复盘工作流、配置样例、Prompt 模板、实验功能预览和社区共建设计，不包含投资建议、收益承诺或代客交易。

| 支持方式 | 支持等级 | 说明 |
| --- | --- | --- |
| 0 RMB | Community | 使用全部社区功能；通过 GitHub 获取更新；可参与 Issue、讨论和路线图反馈。 |
| 轻量支持 | Backer | 获得支持者致谢；优先整理和回应复现清晰的 Issue；共享社区配置样例、Prompt 片段和复盘模板。 |
| 长期支持 | Builder | Backer 权益；优先体验观察仓预警、报告追问、市场资讯面板等实验功能；参与路线图投票和模板共创。 |
| 项目共建 | Partner | 面向插件、数据源适配、报告模板、教程文章和社区案例共建；具体合作以项目阶段和维护精力为准。 |

## 反馈建议

提交问题时建议包含：

- 操作系统和 Python 版本。
- 执行的命令。
- 报错日志或截图。
- 是否修改过配置文件。
- 是否能稳定复现。
