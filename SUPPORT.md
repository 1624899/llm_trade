# 社区支持

LLM-TRADE 社区版不通过 token 锁核心功能。公开仓库以社区协作、问题反馈和贡献说明为主，也接受自愿支持帮助项目持续维护。

## 参与方式

| 类型 | 说明 |
| --- | --- |
| Star | 最简单也最重要的支持，方便更多人发现项目 |
| Issue | 反馈 Bug、使用问题、数据源异常和功能建议 |
| PR | 贡献代码、文档、测试、前端体验和数据源适配 |
| 文档 | 补充安装说明、常见问题、截图和教程 |
| 测试 | 补充复现用例和自动化测试 |

## 支持我们

如果这个项目对你有帮助，欢迎 Star、反馈 Issue、提交 PR，或者请作者喝杯咖啡。社区版核心功能保持开放，支持完全自愿，不包含投资建议、收益承诺或代客交易。

## 反馈建议

提交问题时建议包含：

- 操作系统和 Python 版本。
- 执行的命令。
- 报错日志或截图。
- 是否修改过配置文件。
- 是否能稳定复现。
